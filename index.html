Cloth Simulation

A real-time cloth simulation using GPU compute shaders. Features wind dynamics, structural constraints, and realistic bending.

Key Implementation Points

  • Verlet integration for stable particle updates
  • Structural, shear, and bending constraints (XPBD-style)
  • Wind force with noise-driven gusts and normal-based influence
  • GPU compute shaders with SSBO for particle data

Future Improvements

Plan to add self-collision detection, optimize constraint solving with parallelization, and explore more realistic aerodynamics for wind–cloth interaction.
